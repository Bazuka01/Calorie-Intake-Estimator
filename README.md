# Calorie-Intake-Estimator

A web-based application designed to estimate daily calorie needs based on user-provided health and activity data.

## Features
- Calculate estimated daily calorie intake
- Support for different activity levels
- Weight loss, maintenance, and weight gain recommendations
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Planned Features
- Improved UI/UX
- Input validation
- Unit conversion (lbs/kg, ft/cm)
- Personalized calorie recommendations
- Flask backend integration

## Project Status
Currently in development.

## Goals
This project is being developed to strengthen skills in:
- Frontend development
- JavaScript logic
- Python backend development
- Problem-solving and software design
